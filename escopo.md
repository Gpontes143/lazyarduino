# requisitos minimos

l -> lista placas - arduino-cli board list
u -> compila e upload
c -> compila - arduino-cli compile --fqbn <FQBN> <SKETCH_PATH>
U + shift|U -> upload - arduino-cli upload -p <PORT> --fqbn <FQBN> <SKETCH_PATH>
i + shift|I -> downloand libs
i -> downloand core
s -> serial monitor - arduino-cli monitor -p <PORT>
